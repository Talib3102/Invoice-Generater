# Project Overview: What is this?

This project is a simple but powerful tool to help people **create and manage invoices**.

### Why did I build this?
Managing bills and invoices can be messy. I wanted to create a system that makes it easy to keep track of who owes what, all in one place.

### What can it do?
1. **Create Invoices**: Just type in the details and save it!
2. **List Invoices**: See all your saved invoices in a beautiful list.
3. **Manage Data**: You can delete old invoices to keep your list clean.

### How it works (Simple Version)
- **The Website (Frontend)**: This is what you see. It's built using **React**, which makes it fast and smooth.
- **The Server (Backend)**: This is the "brain" of the project. It handles all the logic using **Java and Spring Boot**.
- **The Database**: This is where all your information is safely stored. We use **MySQL** for this.

### Who is this for?
This is a great example of a **Full-Stack Application**. It's perfect for anyone who wants to see how a modern website talks to a database to manage information!
